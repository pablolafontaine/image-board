# kouchan
100% Rust imageboard backend and frontend using Actix and Yew respectively.

 **WIP**
 
## Setup
1. Install Docker
2. `git clone https://github.com/pablolafontaine/kouchan.git`
3. `cd image-board`
4. `make build`
5. `echo [YOUR MONGODB URI] > api/.mongodb_key`
6. `make up`
